# نظام إدارة المؤسسات - ERP System v2.0

نظام متكامل لإدارة المؤسسات يشمل إدارة الفروع، الموظفين، الطلبات، والتقارير.

## المميزات

- تسجيل دخول آمن مع صلاحيات متعددة (مدير عام، إدارة، موظف)
- إدارة 9 فروع مع عرض الموظفين
- نظام طلبات مع إمكانية التحويل بين الفروع
- لوحة تحكم المدير العام مع الإشعارات
- تصميم متجاوب يعمل على جميع الأجهزة

## هيكل المشروع

```
my-erp-system/
├── src/
│   ├── db.js              # قاعدة البيانات (Supabase)
│   ├── auth.js            # المصادقة
│   ├── components/
│   │   ├── branches.js    # مكون الفروع
│   │   ├── org-chart.js   # الهيكل التنظيمي
│   │   └── requests.js    # مكون الطلبات
│   └── styles.css         # الأنماط
├── api/
│   ├── login.js           # API تسجيل الدخول
│   ├── get-branches.js    # API جلب الفروع
│   └── forward-request.js # API تحويل الطلبات
├── index.html             # الصفحة الرئيسية
├── vercel.json            # تكوين Vercel
├── package.json           # اعتماديات المشروع
└── README.md              # وثائق المشروع
```

## التشغيل

```bash
npm install
npm run dev
```

## النشر على Vercel

```bash
npx vercel --prod
```

## التقنيات المستخدمة

- Alpine.js - إطار عمل الواجهة
- Supabase - قاعدة البيانات والمصادقة
- Vercel - الاستضافة والـ Serverless Functions
- CSS3 - التصميم المتجاوب
